## 🚀 DevStride MCP Server Just Got an Upgrade

For anyone using an AI assistant with DevStride, the MCP connection can now do a lot more — and costs fewer tokens doing it.

### ✨ What's New

- **Most of the product is now reachable.** Before, the assistant could only touch a subset of DevStride. Now it can reach almost anything the API can do — including less common admin and config tasks — always within the permissions you already have.
- **~26 new purpose-built tools**, including delivery metrics (velocity, throughput, cycle time, time-tracking reports), roadmap milestones, sprint/cadence setup, notifications, board management, and git commits/branches linked to an item.
- **It won't silently fall behind.** A new automated check makes sure every DevStride API endpoint is either exposed to the assistant or intentionally set aside with a reason — so as we ship new features, the MCP keeps up automatically.

### ⚡ Why It's Also Leaner

- We slimmed down how tool definitions are described to the assistant, so connecting uses meaningfully fewer tokens.
- There's an optional **"core" mode** that lists just the everyday tools for token-sensitive setups — the full toolset is still reachable, it's just not all shown up front.
