## New Features

### ⏱ Sortable Time Entries

- Time Entries tables are now sortable on each column
- Full keyboard navigation of the Time Entries table
- Long descriptions truncate with an ellipsis; the full description is revealed in a hover tooltip

### 🖱 New Drag in the Map

- Drag operations now display a helpful drag skeleton that follows the mouse cursor during drag
- New bumpers at the top and bottom of a column make it easy to drag to the top of a long list and auto-scroll
- Drag-to-rank intent is now indicated with a thin insert line between rows
- Drag-to-move intent is now indicated with a yellow background on the drop target
- Sped up DOM render during dragging

### 📋 Export Popup

- Select All control added to the header
- Find input added to the header which filters the list

### ⚡ Fast Delete

- From the Map View, users can press **Del** to archive selected rows
- From the Map View, users can press **⇧ Del** to fast delete selected rows

## Quality Improvements

- Users with the Member role can now import templates and duplicate items successfully without crashing the browser
- Import Template now only allows import to a valid parent location
- Import Template now supports name lookups instead of only UUID references
- Unarchive now asks if you want to unarchive children, and correctly unarchives every selected item and its children
- Gantt chart "TODAY" marker elevated to sit on top of all other content
