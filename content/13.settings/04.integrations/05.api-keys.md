---
title: "API Keys"
description: "API keys are managed under My Account → API Keys."
---

API keys are personal credentials managed under **My Account**, not in organization settings — even though they're often used for integrations like the [DevStride MCP server](/devstride-mcp-server/setting-up-your-connection).

➡️ See **[API Keys](/my-account/api-keys)** for creating a key, the one-time secret, and (for administrators) org-wide key management.
