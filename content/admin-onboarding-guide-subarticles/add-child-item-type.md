### Add Child Item Type under Base Item Type

Adding a new child item type under a base item type is a simple process. Follow the steps outlined below:

![ADD_CHILD_ITEM.png](/images/ADD_CHILD_ITEM.png)

### Step 1: Open the New Item Type Form
- Locate and click on the button or link in your system that says "New Item Type" or similar. This will open the form for creating a new item type.

### Step 2: Enter the Label
- In the "LABEL" field, enter the name for your new item type. 
- Make sure that the name you choose is at least 3 characters long, as required by the system.

### Step 3: Select a Parent Item Type
- Click on the "Parent Item Type" dropdown menu.
- A list of existing item types will appear.
- Scroll through the list and select the Base Item Type under which you want to add the new child item type.

### Step 4: Save Your New Item Type
- After selecting the base item type, review the information you've entered.
- Click the "Create Item Type" button at the bottom of the form to save the new item type.

After following these steps, your new child item type should now be added under the selected base item type.
