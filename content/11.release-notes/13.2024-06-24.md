## Enhanced Navigation and Persistent Views

We are excited to bring you the latest updates to DevStride! This release focuses on enhancing usability, improving navigation, and introducing new features to streamline your workflow. Here’s what’s new:

#### Powerful “Search for Anything” Menu

Introducing a robust search menu, accessible via cmd + k on Mac and ctrl + k on PC. This allows you to quickly find anything within the app, boosting productivity and navigation speed.

#### Persistent View State

Your last viewed page state is now saved, ensuring you return to where you left off the next time you visit. This feature enhances continuity and efficiency in your workflow.

#### Inline Code Button in Editor

We've added an inline code button to the editor, making it easier to incorporate code snippets into your documentation and notes. This feature enhances the clarity and readability of your technical content.

#### Current Cycle Date Tag

The date is now visible in the current cycle tag, providing immediate context when viewing cycles. This addition helps you keep track of timelines more effectively.

#### Improved Mobile Usability for Board Page

The board page has been optimized for mobile devices, offering a smoother and more intuitive experience when managing your tasks on the go.

#### Alphabetical Order for Filters

Filters are now displayed in alphabetical order, making it easier to find and apply them.

#### Recent Items in Board Selectors

Board selectors now include recently accessed items, allowing for quicker access to frequently used boards.

#### Custom Field Sorting

We’ve resolved issues related to custom field sorting, especially when multiple option values are involved. Your custom fields will now sort correctly and consistently.
