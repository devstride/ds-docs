## 🤖 DevStride MCP Released

DevStride now ships with a Model Context Protocol (MCP) server, letting you connect AI assistants like Claude directly to your DevStride workspace.

### Getting started

1. Go to your **API Keys** page at `/my-account/api-keys` to generate a new API key
2. Follow the install instructions provided alongside your new key to configure your MCP client

### Token efficiency

We worked hard to make the MCP server token-efficient so you can do more with less context. The server already exposes a wide range of tools covering common DevStride workflows.

### Discover what you can do

Not sure where to start? Just ask your AI assistant:

> **What am I working on?**

It will return a list of every action you can take through the MCP server.

### Phase one — more to come

This is **phase one** of DevStride MCP. We will continue expanding the toolset in upcoming releases.

Please share your feedback on AI performance with the MCP tools — especially around how many tokens it consumes for your workflows. Your input directly shapes how we tune and expand the server in future phases.
