**Features & Enhancements**

- Added a "Due Date" field to work items. Overdue items will now be highlighted in red.
- You can now "mention" Workstreams or Work Items using the hashtag (#) key in descriptions or comments.
- When changing organizations settings, users will now only get a warning if the setting change will edit item field values.
- Added an option in the Cycle reports to include events that happened after the cycle ended.
- General speed improvements.

**Bug Fixes**

- Fixed a bug where users could delete the default custom field collection in settings.
- Fixed activity log items for time estimation.
