## Top toolbar

The top toolbar is for navigation and controls for the weekly timesheet.

| Feature | What it does |
| --- | --- |
| **Week navigation** | Move to the previous or next week. |
| **Today** | Jump to the week that contains today. |
| **Date selection** | Jump to the week for a chosen date. |
| **View mode** | Switch between list view and grouped view. |
| **Weekend toggle** | Show or hide weekend columns. |
| **Export** | Download timesheet data. |
| **Mode selector** | Switch between single-user mode and multi-user mode. |
| **Search and add task** | Find tasks and add them so you can log time against them. |

## Timesheet grid

You enter and edit time in the grid.

- **Rows** — Tasks you are logging time against.
- **Columns** — Days of the week.
- **Cells** — Hours worked for that task on that day.
- **Totals** — Daily and weekly totals are shown for reference.
