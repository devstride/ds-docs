title: "Multi-user timesheets"
description: "Using multi-user mode to view and edit time for more than one person."
---

The plugin can manage timesheets for multiple users.

## Use multi-user mode

1. In the mode dropdown, enable **Multi-user mode**.
2. The grid shows time logged by user for the tasks shown.
3. Select users from the user list as needed.
4. Enter or adjust time for the selected user in the appropriate rows.

Administrators can edit or delete entries for other users when your organization allows it.
