Filters help you focus on the entries you care about instead of scrolling the full list.

## Available filters

You can filter by:

- **Assignee**
- **Items**
- **Workstream**
- **Team**

Several filters can be active at once to refine the view.

## How filter logic works

- **Between filter types** — Conditions use **AND** logic. For example, an assignee filter and a workstream filter together show only entries that match **both**.
- **Within one filter type** — Multiple selections use **OR** logic. For example, several assignees show entries for **any** of those users.

This makes it easier to review time logs and manage timesheet data for specific people, work, or teams.
