The Weekly Logs Plugin is a time-tracking and timesheet management tool. You can log and manage weekly work hours across tasks and projects.

The plugin provides a grid-based interface where you enter time for each task on each day of the week. It supports individual timesheets and multi-user management.

## Key capabilities

**Weekly time tracking for tasks**  
Log time spent on different tasks throughout the week. This helps keep accurate records and shows how time is spread across work.

**Task search and selection**  
Find and select tasks quickly, including with advanced search, so you can add time entries without hunting through long lists.

**Spreadsheet-style entry grid**  
Time appears in a grid similar to a spreadsheet. Enter or update hours in cells, and remove entries you no longer need.

**Multiple view modes (list and grouped)**  
Use **list** view for individual entries, or **grouped** view to organize entries by workstream.

**Notes for each entry**  
Add notes or descriptions on time entries so managers and teammates understand what work was done.

**Multi-user timesheet management**  
View and manage time for multiple users, which supports team-level reporting and oversight.

**Auto-save and draft protection**  
Changes save automatically. Partial entries are kept if you navigate away or the session is interrupted.

**Export timesheet data to CSV**  
Export time logs for reporting, analysis, or use in other tools (for example Excel).

**Built-in validation**  
Entries are validated for correct values and rules, reducing invalid formats, missing required fields, or limits being exceeded.
