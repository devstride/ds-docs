## Add a task to the grid

1. Type the task name or task number in the item search field.
2. Use **Advanced search** if you need to browse by workstream hierarchy.
3. Select the task from the results.
4. Choose **Add** to add all selected tasks.

Each added task appears as a new row in the timesheet grid.

## Enter time

1. Find the task row.
2. Click the cell for the day you want.
3. Enter the hours worked.
4. Press **Enter** to save the entry.

## Add notes or a description

1. Click the notes icon on the time entry cell.
2. In the dialog, enter your notes or description.
3. Save the entry.
