![Weekly Logs export dialog](/images/weekly-logs/export.png)

At the end of the month, or whenever reporting is due, you can export your time entries directly from Weekly Logs.

Click **Export** in the header to open the export dialog. Start by giving the export a name — it defaults to "Timesheet Export" with today's date, but you can change it to anything.

## Date range

Set a **start** and **end** date. Defaults are the first day of the current month through today. The range can span up to a year. This controls which entries are included, regardless of which week you're currently viewing.

## Filters

You can filter what gets exported:

- **Assignees** — export entries for specific team members only.
- **Workstreams** — narrow down to a specific part of your hierarchy.
- **Custom fields** — if your organization uses custom fields, choose which ones to include as columns in the export.

## Download

When everything is set, click **Export**. The file downloads automatically as a CSV — ready to open in Excel or import into any reporting tool.

## What's in the file

The exported CSV includes:

- Date
- Full name
- Workstream path
- Task ID
- Task name
- Work item type
- Any selected custom fields
- Hours logged
- Notes attached to the entry
