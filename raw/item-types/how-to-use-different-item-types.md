# How to Use Different Item Types

> 

Our tool includes various item types to help organize and manage your projects. Below is a list of examples.

## General Items

- **General**
  - This is the default item type for general tasks and notes.

## Task-Related Items

- **Compliance Task**
  - These tasks are related to compliance and regulatory requirements.
- **GTM Org**
  - This category includes tasks pertaining to the Go-To-Market organizational structure.

### Sub-item under GTM Org

- **Sales Task 2**
  - A specific task within the GTM Org category.
- **General & Administrative**
  - This category is for items dealing with the general and administrative operations of the business.

### Sub-item under General & Administrative

- **G&A Task**
  - A task related to general and administrative functions.

## Idea Management

- **Ideas**
  - A category dedicated to the capture and tracking of creative ideas and suggestions.

## Module Development

- **Module**
  - Represents a significant section or component within a project.

### Sub-items under Module

- **Capability**
  - Refers to specific features or abilities that a module possesses.

#### Sub-item under Capability

- **Epic**
  - A substantial work item that contains multiple related tasks.

#### Work Items in an Epic

- **Defect**
  - Identifies bugs or issues that need to be addressed.
- **Story**
  - Provides detailed descriptions of features or tasks from the perspective of the end-user.

When using these items in your workflow, make sure to adhere to the following guidelines:

- **Categorization**: Properly categorize each task or note under the correct item type to maintain clarity and organization.
- **Status Indicators**: Use status indicators to quickly identify the state of each item.
- **Hierarchy**: Respect the hierarchy of items to track progress and dependencies accurately.
- **Accessibility**: Ensure that team members have the appropriate access levels to the items they need to work on.

By following these practices, you can effectively use the different item types to manage your projects and tasks efficiently.
