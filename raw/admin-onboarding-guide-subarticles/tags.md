# Tags

> 

## Data Model Configuration

### Tags

Tags are used for categorizing items for better organization and filtering.

![CREATE_NEW_TAG.png](/images/CREATE_NEW_TAG.png)

#### 1. Create Tags

1. Go to `Settings` > `Data Model` > `Tags`.
2. Click `New` to create a new tag.
3. Enter the name for your tag and choose a color for easy identification.

#### 2. Group into Collections

Group tags into collections for organized management.

1. To create a new collection, click on `New` next to 'Group into Collections'.
2. Drag and drop tags into the created collection to organize them.

#### 3. Assign to Item Types

Assign tags to different item types to enable tagging on items.

1. Choose an item type from the list.
2. Select a collection to assign the tags from that collection to the item type.

Note that tags **Blocked** and **Bug** are reserved by Devstride's system and cannot be changed or removed.

You can delete tags by clicking the red **"X"** button next to each user-defined tag.
