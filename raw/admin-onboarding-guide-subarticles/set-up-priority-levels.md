# Set Up Priority Levels

> 

### Priority Levels

![SET_PRIORITY.png](/images/SET_PRIORITY.png)

Define priority levels to classify the importance of items.

#### 1. Create Levels

1. Go to `Settings` > `Data Model` > `Priority Levels`.
2. Click `New` to add a new priority level.
3. Name the priority level and choose a color.

#### 2. Group into Collections

1. Organize priority levels into collections if required.

#### 3. Assign to Item Types

1. Assign priority levels to specific item types.
