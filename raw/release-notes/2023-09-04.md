# 2023-09-04

> 

## Feature Updates and Enhancements

After focusing on user feedback over the past month, we've implemented several improvements to better meet your needs. Here's what's new:

- **Email Update:** Users can now change their registered email addresses, offering more customization options.
- **Multi-Select Drag:** We've added the ability to select and move multiple rows at once, making list rearrangement easier.
- **Batch Edit for Dependencies:** To simplify the management of interrelated tasks, we've added the capability to edit task dependencies in bulk.
- **Enhanced Sorting:** New sorting fields like Start Date, Author, and Point Effort are now available, providing more control over data views.
- **Team Restoration:** Users can now restore archived teams, offering more flexibility in team management.
- **Default Values for Subtasks:** Subtasks will now inherit default values from their parent tasks, reducing manual input requirements.
- **Subtask Status:** A new status option for subtasks has been added for more detailed tracking and insights.

## Bug Fixes and Stability Improvements

We've also fixed several bugs to improve system stability and user experience. Our focus remains on maintaining a reliable workflow for all users.
