# üöÄ Major UI/UX Modernization Release

Today we are releasing the **largest user experience upgrade ever** across DevStride. This release modernizes interaction patterns, improves performance and accessibility, and delivers a dramatically more fluid experience‚Äîwhile preserving all existing data, workflows, and configurations.

## Executive Summary

This release **rewrites the DevStride user interface** and addresses virtually every UI suggestion submitted by customers and partners over the last four years. Most users will feel the impact immediately through **less friction, fewer clicks, and faster navigation**, even if they never explore individual features in detail.

## Support

If you would like live training on the new release, you can email **phil@devstride.com** directly to set up a time.<br />


If you encounter a bug, please let us know using the new **Support** button in the app.

## Headline Features

- **Login** ‚Äì Animated login page with dark mode detection and polished MFA flows
- **Modals** ‚Äì Completely redesigned modals with drag support and intelligent auto-focus
- **Items Table** ‚Äì Full Excel-like keyboard navigation throughout item tables
- **Gantt** ‚Äì New toolbar, improved navigation, keyboard-accessible data grid, and dependency visualizer
- **Map** ‚Äì Multi-select, new counters, and improved right-click edit options
- **Boards** ‚Äì Better folder navigation, new display options, improved defaults, and faster subtotals
- **Item Drawer** ‚Äì Show/hide and reorder fields, new progress tracker, migrated watchers, and improved focus/disabled states
- **Color Themes** ‚Äì Dynamic theme chooser with categorized previews
- **Create Items with AI** ‚Äì Next-generation AI-powered item creation workflows
- **Dependency Graph** ‚Äì Infinite canvas with pan/zoom, keyboard shortcuts, and color-coded relationships
- **Time Tracking** ‚Äì New time tracking workflows
- **Mobile** ‚Äì Every screen reviewed and optimized for mobile devices
- **Help & Support** ‚Äì New documentation site at **docs.devstride.com** and native support requests
