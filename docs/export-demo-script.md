# Weekly Logs — Export

[▶ Watch Demo](export-demo.mp4)

---

At the end of the month, or whenever reporting is due, you can export your time entries directly from Weekly Logs.

Click the **Export** button in the header to open the export dialog. Start by giving the export a name — it defaults to "Timesheet Export" with today's date, but you can change it to anything.

Next, set the date range. Pick a start and end date — defaults to the start and current day of the current month. The range can span up to a year. This controls which entries are included, regardless of which week you're currently viewing.

You can also filter what gets exported. Use the **Assignees** filter to export entries for specific team members only. Use the **Workstreams** filter to narrow down to a specific part of your hierarchy. And if your organization uses custom fields, you can choose which ones to include as columns in the export.

Once everything is set, click **Export**. The file downloads automatically as a CSV — ready to open in Excel or import into any reporting tool.

The exported file includes: date, full name, workstream path, task ID, task name, work item type, any selected custom fields, hours logged, and any notes attached to the entry.
