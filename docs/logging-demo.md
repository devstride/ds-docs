# Weekly Logs — Logging Time

[▶ Watch Demo](time-logging-demo.mp4)

---

This is where time tracking actually happens. Let's walk through logging hours in Weekly Logs.

Each row in the grid is a task. Each column is a day of the week.

Click a cell. Type your hours. It accepts decimals — so 1.5 meaning 1 hour 30 minutes, 2.25 for 2 hours 15 minutes, 4, and so on are all valid. Hours are rounded to the nearest quarter automatically.

You can log against multiple days for the same task. Just click each day's cell and enter the hours. You don't need to hit Save. The entries you make are automatically saved.

Need to add more context to an entry? Click the notes icon inside the cell — or press **Space Bar** — to open the notes popup. Add a description and save. The note stays attached to that specific entry, on that specific day.

To remove a single entry, clear the cell or set it to zero. To delete all entries for a task across the entire week, click the delete icon in the total cell. Weekly Logs will ask you to confirm before deleting.

That's all there is to it. Click, type, done. The save happens on its own.
