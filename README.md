# DevStride Documentation

Documentation site built with [Docus](https://docus.dev) (Nuxt Content).

## Setup

Install dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```

Generate static site:

```bash
pnpm generate
```

Check out the [Docus documentation](https://docus.dev) for more information.
